# Tkinter draw
Draw vector graphic images and export as tkinter code.

![Christmas tree drawing](examples/img1.png)

Features:
- Draw shapes, triangles, rectangles, circles
- Adjust shape line width, border colour, background colour
- Movement zoom and pan
- Colour picker
- toggle grid lines

Controls:
- Hold `Ctrl` key and click canvas to select vertices of poligon
- `Ctrl + Z` to undo
- `[` or `]` to change line thickness.
- Click and drag to pan
- scroll mouse wheel to zoom
